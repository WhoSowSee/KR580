Меню «Файл» содержит все операции с файлами эмулятора.

**Основные операции**
Новый файл (Ctrl+N) – сброс ОЗУ и регистров. Открыть (Ctrl+O) – загрузка .580 с автоопределением формата. Сохранить (Ctrl+S) – быстрое сохранение в текущий файл. Сохранить как (Ctrl+Shift+S) – сохранение в новый файл.

**Импорт и экспорт**
Импорт (Ctrl+I) – загрузка подпрограммы в текущий сеанс без сброса памяти. Экспорт (Ctrl+E) – сохранение выделенного блока памяти в отдельный файл.

**Старый формат**
Открыть и Сохранить в старом формате – работа с файлами оригинального эмулятора (то же расширение .580, другой внутренний формат). В меню помечены как «старый формат». Доступны только через меню (не с клавиатуры).

При наличии несохранённых изменений появляется диалог подтверждения.
