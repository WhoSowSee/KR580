Сетевой адаптер КР580 обменивается байтами через TCP/IP по порту устройства `03h`.

- Буфер приёма RX отображается шестнадцатеричными строками по 16 байтов, а справа показывается только последнее отправленное значение
- В нижней строке показаны режим, адрес, TCP-порт, состояние соединения и счётчики байтов
- Кнопка с глобусом открывает небольшое окно настройки режима, адреса и порта
- Клиент подключается к указанному узлу, сервер слушает указанный локальный адрес и порт
- Применение настроек безопасно останавливает прежнее соединение и запускает новое только для текущего сеанса
- Стартовые адреса и порты клиента и сервера сохраняются в «Настройки → Общие»; запуск начинается в режиме клиента
- Системные ошибки сокета не выводятся: строка состояния показывает короткое значение, например `Отклонено`, `Тайм-аут` или `Ошибка`
- Кнопка очистки удаляет только содержимое RX/TX, не меняя настройки, статус и состояние соединения; для пустых буферов ничего не происходит
- Окно можно открепить, перемещать за верхнюю полосу и закрепить поверх других окон

`OUT 03h` заменяет последнее отправленное значение и передаёт байт подключённому узлу. `IN 03h` возвращает следующий байт RX или `00h`, если данных нет.
