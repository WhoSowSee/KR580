KR580 – эмулятор микропроцессорной системы на базе КР580ВМ80А, совместимого с Intel 8080.

Приложение показывает регистры, флаги, ОЗУ, машинные циклы и такты процессора. В нём можно вводить программы в hex-кодах, выполнять их по тактам или командам, работать с портами 00h–04h и сохранять результаты в файлы.
