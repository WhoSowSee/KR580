**Вызвать справку** открывает это окно. По умолчанию используются Ctrl+H и F1; действие Ctrl+H можно переназначить.

Поле слева ищет одновременно по названиям и содержимому статей. Кнопка над деревом раскрывает или сворачивает все разделы. Выбор результата очищает поиск и показывает найденную статью.

Текст статьи доступен для выделения и копирования через Ctrl+A/Ctrl+C. Esc, клик по фону или кнопка закрытия закрывают окно.

**О программе** показывает версию приложения и ссылку на репозиторий.
