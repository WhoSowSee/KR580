Меню «Справка».

**Вызвать справку** (Ctrl+H / F1)
Открывает это окно с древовидной структурой: категории слева (кликните на значок ▶ слева от названия категории для раскрытия), содержимое справа. Закрывается Esc или кликом вне диалога.

**О программе**
Информация о версии, описание эмулятора и ссылка на GitHub-репозиторий проекта.
