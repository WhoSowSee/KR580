Настройки внешнего вида (Ctrl+, → вкладка «Внешний вид»).

**Тема оформления**
Список разделён на две группы: «Тёмные» и «Светлые». Текущая схема по умолчанию – Tokyo Night.

Доступные схемы:
• Tokyo Night / Tokyo Night Light
• Черно-белая тёмная / черно-белая светлая
• Kanagawa Wave / Kanagawa Lotus
• Catppuccin Mocha / Catppuccin Latte
• Nord
• Gruvbox Dark / Gruvbox Light
• Material Ocean

В каждой строке слева показано имя темы, справа – увеличенная палитра из основных цветов схемы. Выбор применяется сразу; «Отмена» возвращает прежнюю тему, «Сохранить» записывает её в настройки.
Черно-белая светлая и Catppuccin Latte используют приглушенные светлые поверхности и мягкий текстовый контраст, чтобы не давить яркостью.

**Цветовая индикация**
• Флаги на схеме: зелёный = 1, серый = 0
• Таблица ОЗУ: PC, SP, стек и курсор используют акцентные цвета выбранной схемы
• В черно-белых схемах кнопки внешних устройств, выполнения и сброса используют один монохромный акцент; неактивные лампы приглушаются до серого

**Типографика**
• Моноширинный шрифт для hex-значений
• Пропорциональный шрифт для подписей и текста
