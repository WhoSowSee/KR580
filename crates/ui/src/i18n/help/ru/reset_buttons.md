Кнопки сброса – правый нижний ряд панели управления.

**Сброс ОЗУ** (красная кнопка, Ctrl+Shift+R)
Заполняет все 64 КБ памяти нулями. Программа и данные теряются. Действие можно отменить (Ctrl+Z).

**Сброс регистров** (фиолетовая кнопка, Ctrl+Shift+G)
Обнуляет все программно-доступные регистры: A = 00h, F = 02h, B–H = 00h, SP = 0000h, PC = 0000h. Действие можно отменить.

**Сброс флага HLT** (Ctrl+Shift+H)
Снимает блокировку выполнения после команды HLT, не затрагивая память и регистры.

Все три действия также доступны через меню «МП-Система».
