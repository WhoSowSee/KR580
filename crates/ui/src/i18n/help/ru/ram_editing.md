Панель «Ячейка ОЗУ и её значение» – инструмент редактирования памяти в правой части окна под таблицей ОЗУ.

**Элементы панели**
Поле адреса (4 hex-цифры, 0000h–FFFFh) с кнопками ± для пошаговой навигации. Поле значения (2 hex-цифры, 00h–FFh) с кнопками ±. Кнопка ↵ для применения значения. Кнопка «…» для открытия выпадающего списка всех команд процессора.

**Выпадающий список команд**
244 документированные команды КР580ВМ80А с поиском по hex-коду или мнемонике. Столбцы: код, мнемоника, операнд, длина, тип, адресация. При выборе команды её код записывается в текущую ячейку.

**Горячие клавиши**
Enter в поле адреса – переход к ячейке и фокус на поле значения. Enter в поле значения – применение и переход к следующей ячейке. Ctrl+Enter – поиск hex-шаблона вперёд от текущего адреса.
