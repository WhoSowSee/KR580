Панель «Ячейка ОЗУ и её значение» – инструмент редактирования памяти в правой части окна под таблицей ОЗУ.

**Элементы панели**
Поле адреса (4 hex-цифры, 0000h–FFFFh) с кнопками ± для пошаговой навигации. Поле значения (2 hex-цифры, 00h–FFh) с кнопками ±. Ввод в поле значения сразу отображается в выбранной строке таблицы ОЗУ, но записывается в память только после ↵. Ctrl+V с HEX-парами через пробел записывает последовательность с выбранной строки даже без режима редактирования. Кнопка ↵ применяет значение. Кнопка «…» открывает выпадающий список всех команд процессора.

**Выпадающий список команд**
244 документированные команды КР580ВМ80А с поиском по hex-коду или мнемонике. Столбцы: код, мнемоника, операнд, длина, тип, адресация. Стрелки ↑/↓ и Tab/Shift+Tab перемещают выделение по найденным командам, Enter записывает выделенный код в текущую ячейку.

**Горячие клавиши**
Enter в поле адреса – переход к ячейке и фокус на поле значения. Enter в поле значения – применение и переход к следующей ячейке. Ctrl+Enter – поиск hex-шаблона вперёд от текущего адреса.
