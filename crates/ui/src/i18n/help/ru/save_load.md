Сохранение и загрузка полного состояния эмулятора.

**Сохранение**
Ctrl+S – быстрое сохранение в текущий файл (если файл не выбран – запрашивает путь). Ctrl+Shift+S – «Сохранить как», всегда открывает диалог выбора пути.

**Загрузка**
Ctrl+O – открыть сохранение .580 с автоопределением формата. После загрузки обновляются структурная схема, таблица ОЗУ и строка статуса.

**Старый формат**
Сохранение и открытие в старом формате .580 для совместимости с файлами оригинального эмулятора. Только через меню «Файл».

**Новый файл** (Ctrl+N)
Обнуляет ОЗУ и регистры, создавая пустой сеанс. При наличии несохранённых изменений появляется диалог подтверждения.
