Файл .580 хранит полный образ ОЗУ размером 64 КБ и значение PC. Остальные регистры, флаги, состояние HLT, прерывания и устройства в этот файл не входят; после открытия они принимают начальные значения.

Ctrl+O открывает .580. Ctrl+S сохраняет по текущему пути или запрашивает путь при первом сохранении. Ctrl+Shift+S всегда сохраняет под новым именем.

Ctrl+N создаёт пустой сеанс: ОЗУ заполняется нулями, процессор сбрасывается. При несохранённых изменениях перед открытием, созданием или закрытием появляется подтверждение.

Для обмена выбранными регистрами, флагами и частями ОЗУ используйте импорт и экспорт TXT/XLSX.
