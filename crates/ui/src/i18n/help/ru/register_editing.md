Панель «Регистр и его значение» – выбор и редактирование любого программно-доступного регистра.

**Элементы панели**
Выпадающий список регистров: A, F, B, C, D, E, H, L, SP, PC (стрелки ↑/↓ или клик для выбора). Поле значения – 2 или 4 hex-цифры в зависимости от разрядности регистра. Кнопки ± для пошагового изменения на ±1. Кнопка ↵ для применения.

**Inline-редактирование на структурной схеме**
Наведите курсор на регистр (подсветится), кликните – появится поле ввода. Введите значение и нажмите Enter для применения, Esc для отмены. Tab/Shift+Tab – последовательный переход к соседнему регистру. Стрелки ←/→ перемещают редактирование между аккумулятором и буферными регистрами. В мультиплексоре стрелки ←/→ перемещаются между столбцами, ↑/↓ – между строками. Ctrl+стрелки работают так же. Режим замены переносится на новый регистр.

Все изменения регистрируются в Undo-истории (Ctrl+Z для отмены).
