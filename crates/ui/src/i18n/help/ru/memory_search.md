Поиск hex-шаблона в памяти – быстрый способ найти нужный участок кода или данных.

**Поиск вперёд**
Введите hex-шаблон в поле адреса (например «C3» или «3E06») и нажмите Ctrl+Enter. Эмулятор найдёт первое вхождение, начиная с текущего адреса, и выделит его.

**Поиск назад**
Тот же шаблон + Ctrl+Shift+Enter. Поиск в сторону уменьшения адреса.

**Прямой переход**
Alt+Enter – переход по введённому адресу без поиска.

**Результаты**
• Успех: статус «Найден шаблон по адресу XXXX»
• Неудача: «Нет адресов, соответствующих шаблону»

Совет: для поиска последовательности команд указывайте их hex-коды подряд, например «C33412» для JMP 1234h.
