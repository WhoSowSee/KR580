Дисковод КР580 эмулирует устройство чтения/записи гибких дисков.

**Интерфейс**
• Доступ через панель «Быстрый доступ» (иконка дискеты) или Ctrl+F
• Отдельное окно показывает принятые байты буфера накопителя
• Кнопка образа подключает файл дискеты к порту 01h
• Кнопка сохранения записывает текущий видимый буфер через отдельные варианты .kpd, .img и .bin; по умолчанию выбран .kpd
• Кнопка отключения отсоединяет файл образа, не очищая видимый буфер
• Кнопка двоичного просмотра переключает окно на содержимое файла образа
• Debug-режим пишет байты прямо в видимый буфер без файла образа
• Кнопка очистки сбрасывает только видимый буфер окна
• Обмен данными – через выделенные порты ввода-вывода
• Если устройство не готово, байт отклоняется и в буфер не попадает

**Настройка**
Файл образа выбирается кнопкой в окне дисковода. Последний выбранный путь сохраняется в настройках.
