Дисковод КР580 эмулирует устройство чтения/записи гибких дисков.

**Интерфейс**
• Доступ через панель «Быстрый доступ» (иконка дискеты)
• Отдельное окно показывает содержимое буфера накопителя
• Обмен данными – через выделенные порты ввода-вывода

**Настройка**
Каталог для эмуляции внешнего накопителя задаётся в настройках программы (Ctrl+, → Настройки).
