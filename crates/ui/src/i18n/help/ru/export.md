Экспорт (Ctrl+E) сохраняет состояние в отдельный файл.

В окне экспорта выберите вкладку MS Excel (.xlsx) или Текстовый файл (.txt), задайте страницу Excel или раздел текстового файла, диапазон ОЗУ, нужные колонки, регистры и флаги. Новые страницы и разделы доступны только в текущем сеансе приложения. В Excel каждая добавленная страница хранит свои параметры и записывается отдельным листом в одном .xlsx файле. В текстовом формате каждый добавленный раздел хранит свои параметры и записывается как именованный блок в одном .txt файле. Затем нажмите Экспорт и укажите путь для сохранения. Экспортированный файл можно загрузить в другой сеанс через Импорт (Ctrl+I).

Применение: выделение подпрограммы из большого проекта, сохранение таблиц данных, обмен фрагментами кода между разными сеансами.
