Экспорт данных (Ctrl+E) сохраняет выделенный диапазон ячеек ОЗУ в отдельный файл.

Выберите начальный и конечный адрес диапазона, затем укажите путь для сохранения. Экспортированный файл можно загрузить в другой сеанс через Импорт (Ctrl+I).

Применение: выделение подпрограммы из большого проекта, сохранение таблиц данных, обмен фрагментами кода между разными сеансами.
