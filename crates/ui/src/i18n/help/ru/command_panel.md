Панель системных команд отображает состояние выполнения на левой панели структурной схемы.

**Счётчики и индикаторы**
Номер машинного цикла (M1–M5) – текущий шаг команды. Номер такта (T1–T5) – такт внутри цикла. Фаза – такт инструкции с отсчётом от нуля. Общий счётчик тактов – с начала сеанса. Такт инструкции – номер такта внутри текущей команды.

**Индикаторы микроопераций**
Показывают текущие действия процессора: чтение памяти, запись, операция АЛУ и другие сигналы управления.

**Дешифратор команд**
Отображает код операции, мнемонику, тип команды и режим адресации.
