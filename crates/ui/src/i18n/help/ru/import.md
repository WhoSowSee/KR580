Импорт (Ctrl+I) накладывает данные из TXT или XLSX на текущий сеанс.

Выберите файл, затем лист XLSX или именованный раздел TXT. Если TXT не содержит разделов, используется весь файл. Импортируются только присутствующие в выбранной части ячейки ОЗУ, регистры и флаги; остальные значения не меняются.

Перед импортом поверх несохранённых изменений приложение запрашивает подтверждение. Некорректные hex-значения, неизвестные регистры и повреждённая структура файла отклоняются с сообщением об ошибке.
