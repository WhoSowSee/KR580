Импорт (Ctrl+I) загружает данные из TXT или XLSX в текущий сеанс без полного сброса состояния.

В окне импорта сначала выберите файл. Для XLSX-файла появится выбор листа, если в книге есть листы. Для TXT-файла появится выбор раздела, если файл был сохранён как несколько именованных блоков. Если в TXT нет отдельных разделов, импортируется весь файл.

После подтверждения импортированные регистры, флаги и ячейки ОЗУ записываются поверх текущих значений. Остальная память остаётся без изменений.
