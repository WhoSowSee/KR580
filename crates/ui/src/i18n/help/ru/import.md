Импорт подпрограммы (Ctrl+I) загружает содержимое файла в текущий сеанс без сброса памяти и регистров.

Открывается диалог выбора файла, затем запрашивается адрес для размещения данных. Содержимое файла записывается в ОЗУ начиная с указанного адреса. Остальная память и регистры не изменяются.

Позволяет компоновать программу из нескольких частей – например, основная программа плюс библиотека подпрограмм. Действие можно отменить (Ctrl+Z).
