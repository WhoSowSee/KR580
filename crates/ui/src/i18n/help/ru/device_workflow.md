Работа с внешними устройствами эмулятора.

**Панель «Быстрый доступ»** (внизу левой панели)
Содержит кнопки: Монитор (Ctrl+M), Дисковод, HDD, Сеть, Принтер. Клик по кнопке открывает окно соответствующего устройства.

**Окна устройств**
Открываются поверх главного окна. Закрываются клавишей Esc или кнопкой закрытия в заголовке окна.

**Взаимодействие программы с устройствами**
Доступ через порты ввода-вывода командами IN/OUT. При выполнении программы обращения к портам автоматически направляются в соответствующие окна устройств. Параметры устройств (пути к каталогам, хост, порт) настраиваются в окне настроек (Ctrl+,).
