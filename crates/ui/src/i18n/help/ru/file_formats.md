Форматы файлов эмулятора.

**.580** – основной бинарный формат. Содержит версию формата, полный образ ОЗУ (64 КБ), все регистры (A, F, B–H, SP, PC), флаги, состояния устройств, метаданные. Обратная совместимость между версиями.

**Старый формат** (.580, другой magic) – наследие оригинального эмулятора. Содержит образ ОЗУ и значения регистров. Ограниченная совместимость. Доступен через меню «Файл» с пометкой «старый формат».

**Импортируемые форматы** – любые бинарные файлы (.bin, .com, .hex). Загружаются как есть по указанному адресу. Позволяют использовать программы, скомпилированные внешними ассемблерами.

**Undo/Redo** – Ctrl+Z / Ctrl+Shift+Z, глубина истории 256 шагов. Отслеживаются редактирование регистров, памяти, сброс и импорт. Последовательные изменения одного поля группируются в одну запись.
