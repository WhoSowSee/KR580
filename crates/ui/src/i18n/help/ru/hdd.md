Жёсткий диск КР580 – эмуляция постоянного накопителя большой ёмкости.

**Интерфейс**
• Доступ через панель «Быстрый доступ» (иконка HDD)
• Окно устройства показывает принятые байты буфера
• Кнопка отделения открывает буфер как самостоятельное окно без системной рамки; его можно перемещать за пустую область заголовка
• В отдельном окне кнопка закрепления удерживает буфер поверх остальных окон, а кнопка возврата снова встраивает его в эмулятор
• Обмен данными – через порт ввода-вывода 02h

**Элементы окна**
• Выбор каталога – указывает папку; создаёт/открывает `hdd.kpd` внутри неё (только на текущий сеанс)
• Просмотр содержимого файла – переключает окно между буфером записи и содержимым файла на диске
• Debug-режим – пишет байты прямо в видимый буфер без файла
• Очистить буфер – сбрасывает только видимый буфер окна
• Удалить файл – удаляет `hdd.kpd` с диска (с диалогом подтверждения)
• Создать файл – создаёт `hdd.kpd` в текущем или стандартном каталоге

**Настройка**
Каталог по умолчанию для файла HDD задаётся в Настройки → Общие → Каталог HDD.
Если не указан – используется домашняя папка пользователя. При выборе каталога
выполняется проверка прав на запись – если запись невозможна, показывается уведомление об ошибке.

Обмен данными – через инструкции `OUT 02h` и `IN 02h`.
Если устройство не готово, байт отклоняется и в буфер не попадает.
