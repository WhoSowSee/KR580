Жёсткий диск КР580 – эмуляция постоянного накопителя.

**Интерфейс**
• Доступ через панель «Быстрый доступ» (иконка HDD)
• Окно устройства показывает содержимое буфера обмена
• Обмен данными – через порты ввода-вывода

**Настройка**
Каталог для виртуального жёсткого диска указывается в настройках программы.
