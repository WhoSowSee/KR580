Таблица содержимого ОЗУ – основной инструмент работы с памятью. Три столбца: Адрес (hex, 0000h–FFFFh), Значение (hex, 00h–FFh), Команда (мнемоника ассемблера).

**Цветовые выделения**
Синий – выбранная ячейка (курсор или текущий адрес в панели редактора). Зелёный – ячейка по адресу PC (счётчика команд), перемещается при выполнении. Коричневый – ячейка по адресу SP (указателя стека). Жёлтый – все ячейки с адресом ≥ SP (стековая область).

**Навигация**
Мышь или стрелки ↑/↓ для выбора строки; адрес синхронизируется с панелью редактора. PageUp/PageDown – прокрутка на ±16 строк. Enter на строке – переход к inline-редактированию значения.

**Выпадающий список команд** (кнопка «…»)
Содержит все 244 документированные команды. Поиск по hex-коду («3E») или мнемонике («MVI A»). Стрелки ↑/↓ и Tab/Shift+Tab перемещают выделение по найденным строкам, Enter записывает выделенный код в текущую ячейку. Столбцы: код, мнемоника, операнд, длина, тип, адресация.
