**Top bar**
File, MP-System, View, Settings, and Help expose the main actions. The CPU icon hides or shows the menu category names. Window controls are on the right.

**Structural diagram**
The left side shows registers, flags, buses, the decoder, machine cycles, and T-states. Register values can be edited directly on the diagram. Device shortcuts and execution speed are below it.

**RAM and controls**
The right side contains the RAM list, memory and register editors, and execution/reset buttons. The selected cell, PC, SP, and instruction operands have distinct highlight states.

**Status bar**
The bottom line reports the latest action, current file path, HLT state, input errors, and device errors.
