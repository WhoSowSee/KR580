Keyboard Shortcuts:

These are the default shortcuts. They can be changed in Settings > Shortcuts. Letter shortcuts use the physical QWERTY key, so English and Russian layouts are equivalent: E/У opens the opcode picker, Alt+Q/Alt+Й jumps to 0000, Alt+E/Alt+У jumps to FFFF, Ctrl+S/Ctrl+Ы saves, Ctrl+M/Ctrl+Ь opens the monitor, Ctrl+F/Ctrl+А opens the floppy buffer, Ctrl+A/Ctrl+Ф opens the network adapter when no text field has captured the key, Ctrl+P/Ctrl+З opens the printer.

- File: Ctrl+N (New), Ctrl+O (Open), Ctrl+S (Save), Ctrl+Shift+S (Save As), Ctrl+I (Import), Ctrl+E (Export)

- Execution: Ctrl+R (Run/Pause), Ctrl+T (Step Instruction), Ctrl+Y (Step Tact), Ctrl+Shift+R (Reset RAM), Ctrl+Shift+G (Reset Registers), Ctrl+Shift+H (Clear HLT)

- Navigation: Tab/Shift+Tab (fields), Up/Down (memory cells and multiplexer rows), Left/Right (schematic registers and multiplexer columns), PageUp/PageDown (+/-16 rows), Alt+Q (0000), Alt+E (FFFF), Ctrl+Arrows (equivalent register-grid navigation). In the opcode picker, Up/Down and Tab/Shift+Tab move the highlighted command.

- Editing: Ctrl+A (Select All inside focused text fields), Ctrl+Z (Undo), Ctrl+Shift+Z (Redo), Ctrl+Enter (search forward), Ctrl+Shift+Enter (search backward), Alt+Enter (jump to 16-bit address operand / device port), Alt+Shift+Enter (return from the last 16-bit address operand jump)

- Windows: Ctrl+M (Monitor), Ctrl+F (Floppy buffer), Ctrl+A (Network adapter outside focused text fields), Ctrl+P (Printer), Ctrl+Shift+C (Stack memory), Ctrl+, (Settings), Ctrl+H/F1 (Help), Esc (close/clear focus)
