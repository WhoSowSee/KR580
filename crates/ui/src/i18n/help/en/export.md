Export (Ctrl+E) saves the state to a separate file.

In the export window, select the MS Excel (.xlsx) or Text file (.txt)
tab, then choose the Excel page or text-file section, RAM range, memory
columns, registers, and flags. New pages and sections live only in the
current app session. In Excel format, each added page keeps its own
settings and is written as a separate worksheet in one .xlsx file. In
text format, each added section keeps its own settings and is written as
a separate named block in one .txt file. Press Export and choose
the save path. The exported file can be loaded via Import (Ctrl+I) in
another session.

Useful for:
- Extracting subroutines from large projects
- Saving data tables
- Sharing code between different snapshots
