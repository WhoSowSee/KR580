**Execution**
• Run one T-state, one instruction, or continuously
• Select 5, 20, 120, or 1000 instructions per second
• Optionally keep the RAM list following PC

**Editing and navigation**
• Edit RAM and registers from panels or the structural diagram
• Search 244 documented opcode encodings
• Paste hexadecimal byte sequences, use Undo/Redo, and follow operands

**Files and data exchange**
• .580 program files containing RAM and PC
• TXT/XLSX import and export with selectable memory, registers, and flags

**Devices and interface**
• Monitor, floppy, HDD, TCP network, and system printer
• Russian and English languages with light and dark themes
• Settings/help search and 27 reassignable actions
