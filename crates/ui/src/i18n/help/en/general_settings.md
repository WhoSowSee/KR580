General Settings (Ctrl+, > General tab):

- Interface language: Russian or English. Applied immediately to menus, labels, status, help.

- Default speed for new sessions:
Slow (~1 instr/sec) - step-by-step observation
Medium (~10 instr/sec) - balanced
Fast (~100 instr/sec) - for working programs
Max - maximum emulation performance
Speed can be changed anytime via the Speed panel.

Reset button restores defaults (Russian, medium speed). Settings persist between sessions.
