The View menu opens the monitor, floppy, HDD, network adapter, and printer windows. The same windows are available from Quick Access and reassignable shortcuts.

**Stack memory area**
Show stack memory area restricts the RAM list to the final 256 bytes, FF00h–FFFFh. The item then changes to Hide stack memory area.

Select it again, press Ctrl+Shift+C, or press Esc to restore the normal list and its previous position.
